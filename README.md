# NoteMind AI

NoteMind AI is a full-stack AI-powered notes application built with React, Vite, TypeScript, Spring Boot, PostgreSQL, and AI API integration.

## Tech Stack

### Frontend

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Router
- TanStack Query

### Backend

- Java
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA

### Database

- PostgreSQL

### AI

- Gemini API first
- OpenAI API support planned later

## Planned Features

- User registration and login
- Create, edit, delete, and search notes
- Organize notes using subjects and tags
- AI note summarization
- AI flashcard generation
- AI quiz generation
- Save AI outputs
- Dashboard with recent notes
- Future RAG-based "Ask My Notes" feature
